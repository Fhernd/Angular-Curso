Angular con Bootstrap 5.2.2

Bootstrap 5 es un framework de CSS que nos facilita y estandariza el desarrollo de sitios web.
Se ha implementado pensando que se adapte tanto a las pantallas de equipos de escritorio como a móviles y tablets.

Bootstrap 5 ha sido desarrollada y es mantenida por la empresa Twitter y la ha liberado como un producto Open Source.
